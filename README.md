# Análise de Algoritmos - 2026/1

Repositório destinado ao armazenamento e organização das atividades, deveres de casa e projetos da disciplina de **Análise de Algoritmo**, ministrada pelo Professor Pablo Coelho Ferreira no IESB.

## Visão Geral
A disciplina aborda a importância de algoritmos eficientes, análise de custos, notação assintótica (O, Ω, Θ) e métodos de resolução de recorrências.

## Estrutura do Repositório
Conforme as diretrizes da aula 01, o projeto deve seguir esta organização de pastas:

```text
a-algo-2026-1/
├── deveres-de-casa/
│   ├── dever-01/
│   ├── dever-02/
│   └── ...
├── trabalhos/
│   ├── trabalho-01/
│   └── trabalho-02/
└── README.md
```

## Convenções de Código
Para manter o padrão de qualidade exigido, utilizamos as seguintes normas:

- **Linguagem**: Python.
- **Estilo**: PEP8 (snake_case para variáveis e funções; PascalCase para classes).
- **Documentação**: PEP257 (Docstrings).
